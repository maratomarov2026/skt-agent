# AI-Агент КФ «Samruk-Kazyna Trust»

Инструмент для анализа заявок и мониторинга благотворительных проектов на основе Claude AI.

## Требования

- Node.js 18+
- Ключ Anthropic API

## Установка

```bash
cd agent
npm install
```

## Настройка

Откройте файл `.env` и вставьте ваш API-ключ:

```
ANTHROPIC_API_KEY=sk-ant-...
```

Получить ключ можно на https://console.anthropic.com/

## Запуск

```bash
npm start
```

## Модули

| Модуль | Назначение |
|--------|-----------|
| А | Проверка комплектности пакета заявки (17 документов) |
| Б | Содержательный анализ проекта + верификация статистики через stat.gov.kz |
| В | Комплаенс-анализ (аффилированность, санкции, реестры) |
| Г | Мониторинг запроса транша — сверка с утверждённой сметой |
| Д | Анализ квартального отчёта пользователя (3 блока) |
| Е | Формирование консолидированного отчёта СППУ |
| Ж | Готовые уведомления заявителю (5 шаблонов) |

## Команды в интерфейсе

| Команда | Действие |
|---------|---------|
| `/модуль А` | Активировать Модуль А (А–Ж) |
| `/файл путь/к/файлу.pdf` | Загрузить документ для анализа |
| `/файлы` | Список загруженных документов |
| `/проект Название` | Задать название текущего проекта |
| `/очистить` | Сбросить историю диалога |
| `/выход` | Завершить работу |

## Поддерживаемые форматы документов

PDF, DOCX, DOC, XLSX, XLS, TXT, CSV

## Пример работы

```
/модуль А
/файл C:\Заявки\ОФ_Болашак\устав.pdf
/файл C:\Заявки\ОФ_Болашак\смета.xlsx
/файл C:\Заявки\ОФ_Болашак\обращение.docx
/проект Школа будущего 2025

Проверь комплектность пакета документов по заявке ОФ «Болашак»
```
