AI-Агент КФ «Samruk-Kazyna Trust»

Автоматизация полного цикла работы с заявками на получение благотворительной помощи — от проверки документов до консолидированного отчёта.

GitHub → Модули
Функциональность

7 модулей — полный цикл заявки

Каждый модуль соответствует конкретному этапу работы согласно Правилам оказания и мониторинга благотворительности КФ.

МОДУЛЬ А

Комплектность пакета

Проверка 17 обязательных документов, сроков справок, подписей и печатей.

МОДУЛЬ Б

Содержательный анализ

Оценка по 6 критериям СППУ. Верификация статистики через stat.gov.kz.

МОДУЛЬ В

Комплаенс-анализ

Проверка аффилированности, санкционных списков, реестров недобросовестных участников.

МОДУЛЬ Г

Мониторинг траншей

Контроль расходов по статьям сметы. Детектирование дробления платежей.

МОДУЛЬ Д

Квартальный отчёт

Анализ финансов, плана инфосопровождения и эффективности проекта.

МОДУЛЬ Е

Консолидированный отчёт

Формирование отчёта СППУ и СПКС (Приложения №18, №19).

МОДУЛЬ Ж

Уведомления заявителю

5 готовых шаблонов: после анализа заявки и после решения Попечительского совета.

Установка и запуск

# 1. Клонировать репозиторий
git clone https://github.com/maratomarov2026/skt-agent.git
cd skt-agent

# 2. Установить зависимости
npm install

# 3. Добавить API-ключ в .env
ANTHROPIC_API_KEY=sk-ant-...

# 4. Запустить агент
npm start
